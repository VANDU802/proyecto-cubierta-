GLOSARIO PENSAMIENTO ALGORITMICO

Por: Duvan Rubiano, Danilo Galvis



A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con pensamiento algoritmico


ALGORITMO

Un algoritmo es un conjunto de pasos o instrucciones ordenadas que se siguen para resolver un problema o realizar una tarea específica. Los algoritmos son fundamentales en la programación de algún trabajo, ya que definen cómo se debe proceder para obtener un resultado. De ejemplo se podría poner el resultado que se obtiene de un problema matemático



HTML

HTML es HyperText Markup Language en español (Lenguaje de Marcado de Hipertexto), se utiliza para crear y estructurar contenido en la web. HTML define la estructura y el formato de una página web a través de una serie de elementos que describen el contenido, como texto, imágenes o enlaces..



NEMOTECNICA

Es un conjunto de técnicas y métodos utilizados para mejorar la memoria y facilitar el recuerdo de información. Estas técnicas suelen basarse en la asociación de la información que se desea recordar con imágenes, palabras o frases que son más fáciles de memorizar.



SEUDOCODIGO

Es la forma de definir un algoritmo o procesos que sean comprensibles para cualquier persona que quiera leerlo y es un lenguaje de programación, Se utiliza para planificar y conceptualizar la lógica de un algoritmo antes de implementarlo en un lenguaje de programación.



SECUENCIAL

Es un tipo de flujo de control en el que las instrucciones se ejecutan una tras otra, en el orden en que aparecen. Es el tipo más básico y directo de control en programación.



CONDICIONAL

Es una estructura de control que permite tomar decisiones y ejecutar diferentes bloques de código basados en una condición específica. Los condicionales permiten que el programa responda a diferentes situaciones y datos de entrada de manera dinámica.



VARIABLE

Es un espacio de almacenamiento que se usa para guardar datos en un computador, las variables tienen cada un nombre para ejecutar un programa.



ISP (PROVEEDOR DE SERVICIOS DE INTERNET):

Un ISP es una empresa u organización que ofrece servicios de acceso a internet, proporcionan la estructura y los servicios necesarios para que los usuarios puedan acceder y utilizar la red.



ROUTER:

Dispositivo que comparte las señales recibidas por el modem a través de cable o señales inalámbricas con wifi por toda la casa, la forma que funciona el router es conectar todos sus dispositivos a su red domestica o a su red wifi la cual permite que sus dispositivos se conecten a internet.



MODEM

Dispositivo que conecta tu casa con tu proveedor de servicios de internet y traduce las señales digitales para que tu ordenador, tu móvil o cualquier aparato que pueda conectarse a internet pueda entenderlas



SWICH

es una estructura de control que permite seleccionar una de varias posibles rutas de ejecución en función del valor de una expresión.



ETHERNET

Es una tecnología fundamental en redes de comunicación de datos. Ethernet es un estándar para redes de área local (LAN) que define cómo los dispositivos en una red deben formatear, transmitir y recibir datos



INTRANET

Es una red privada utilizada dentro de una organización para la comunicación y el intercambio de información entre sus miembros.



INTERNET

Es una red global de redes que conecta millones de computadoras y dispositivos alrededor del mundo, permitiendo el intercambio de información y la comunicación entre ellos.



CABLE (UTP)

Es definido como (Unshielded Twisted Pair, o Par Trenzado No Blindado) es un tipo de cable de red comúnmente utilizado en redes de comunicaciones, incluyendo redes de área local (LAN) y sistemas de telefonía.)
